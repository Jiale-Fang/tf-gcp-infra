# tf-gcp-infra
Terraform template

GCP Api need to be enabled:
- Compute Engine API
- Service Networking API
- Cloud DNS API
- Cloud Logging API
- Cloud Build API
- Cloud Functions API
- Cloud Pub/Sub API
- Eventarc API
- Cloud Run Admin API
- Serverless VPC Access API
- Cloud Key Management Service (KMS) API
- Cloud SQL Admin
